## Setup

1. Clone the repository
2. Open `index.html` in a web browser
3. Configure Telegram bot credentials in `script.js`:
   - Replace `TELEGRAM_BOT_TOKEN` with your bot token
   - Replace `TELEGRAM_CHAT_ID` with your chat ID

## Telegram Bot Setup

1. Create a bot using [@BotFather](https://t.me/BotFather) on Telegram
2. Get your bot token and chat ID
3. Update the credentials in `script.js`
4. Start a conversation with your bot on Telegram
